Composition Failure: M08 -> M09

The morphism from Meta-Prompting to Multi-Agent Systems is the weakest in the chain. Meta-prompted skill files are natural inputs to multi-agent orchestrators (generating specialized sub-agent configurations), but this compositional relationship is not made explicit in the curriculum. Gap Analysis
MUST
Token Economics and Cost Management
No module covers API call costs, context window limits, budget strategies, or cost monitoring. Every production deployment requires this knowledge. A senior architect would expect this in M02 (foundations) or M12 (production). Estimated effort: 1 new section (1,500-2,000 words).
MUST
Expand Modules 04-09 Content Depth
The content cliff between M01-M03 (avg 9,657 words, 22.3 interactive elements) and M04-M09 (avg 5,281 words, 11.5 elements) will be noticeable to students. Priority targets: M08 (4,267 words), M04 (6,144 words), M05 (4,938 words). Estimated effort: 15,000-20,000 additional words across 6 modules.
SHOULD
HTML Diagrams for Modules 06-12
24 standalone HTML diagrams exist for M01-M05 (era-diagram, collaboration-model, m02-*, m03-*, m04-*, m05-*). Zero exist for M06-M12. Visual learning support drops entirely in the second half. Estimated effort: 20-28 diagram HTML files.
SHOULD
Concrete Prompt Injection Defense Code
Module 10 covers prompt injection conceptually (trust hierarchy, CLAUDE.md directives) but lacks concrete code-level defenses: input sanitization, output validation, sandboxed execution contexts. A senior security engineer would expect working examples.
SHOULD
M08 to M09 Compositional Bridge
Add a section showing how meta-prompted skill files feed into multi-agent orchestrators. Example: use meta-prompting to generate four specialized review skills, then orchestrate them with fan-out/fan-in from M09. Closes the weakest link in the module chain.
SHOULD
Claude SDK Programmatic API Coverage
The course teaches Claude Code CLI extensively but does not cover the programmatic Claude SDK for building agent applications. Production multi-agent systems often use the SDK, not CLI orchestration. This gap means students can use agents but cannot build agent systems.
NICE
Context7 Early Introduction
Context7 appears only in M11 but would benefit from a brief mention in M05 (MCP Architecture) as a real-world MCP server example. This removes it from the orphaned concept list.
NICE
JSON Schema and Zod Prerequisites
Both are used extensively in M05-M06 without formal introduction. A brief "Prerequisites" callout or link to resources would help students unfamiliar with these technologies.
