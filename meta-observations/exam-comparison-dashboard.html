Section 4: Gap Analysis

Exam Topics the Course DOESN'T Cover

Agent SDK Hooks (PostToolUse, tool call interception)
Exam Task 1.5 tests knowledge of programmatic hooks for intercepting tool calls, normalizing data formats, and enforcing business rules deterministically. The course teaches approval gates (M10) but not the specific Agent SDK hook API.
CRITICAL GAP 27% domain
Message Batches API & Batch Processing
Exam Task 4.5 covers the Message Batches API (50% cost savings, 24h window, custom_id correlation, no multi-turn tool calling). Course does not mention batch processing at all.
CRITICAL GAP 20% domain
fork_session and Named Session Management
Exam Task 1.7 tests --resume with session names, fork_session for parallel exploration, and when to start fresh vs resume. Course mentions --continue but not the full session management API.
MODERATE GAP
.claude/rules/ with YAML Frontmatter Path Scoping
Exam Task 3.3 tests path-specific rules files with YAML paths fields and glob patterns. Course mentions CLAUDE.md hierarchy but not the dedicated rules/ directory with conditional loading.
MODERATE GAP
Confidence Calibration & Stratified Sampling
Exam Task 5.5 covers field-level confidence scores, calibration with labeled validation sets, and stratified random sampling. Course covers human-in-the-loop but not quantitative confidence calibration.
MODERATE GAP
tool_choice Configuration Details
Exam Tasks 2.3 and 4.3 test "auto" vs "any" vs forced tool selection semantics. Course teaches tool schema design but not the API-level tool_choice parameter configuration.
MODERATE GAP
CI/CD Specific Patterns (--json-schema, session isolation for reviews)
Exam Task 3.6 covers CI-specific patterns: -p flag, --output-format json with --json-schema, session context isolation for independent review, and prior review findings injection. Course mentions claude -p but doesn't go deep on CI patterns.
MODERATE GAP

Course's Competitive Advantage Beyond the Exam

Mental Model Superiority

The PRAO Loop and TCEF Pattern give course graduates a structured reasoning framework that exam-only preparation lacks. These transferable meta-skills accelerate learning in any new Claude feature or API surface.

Hands-On Lab Practice

9 interactive HTML labs with 116 exercises provide practical muscle memory. The exam tests conceptual knowledge; the course builds operational skill. Graduates can DO, not just KNOW.

Production Operations Coverage

Modules 10-12 cover security, deployment, incident response, and maintenance -- real-world concerns that the exam's architecture focus does not address. This is where certifications end and production success begins.

Meta-Prompting & Self-Improving Systems

Module 08's meta-prompting curriculum is entirely absent from the exam. This is a force-multiplier skill: graduates can use Claude to generate, evaluate, and improve prompts at scale -- something the exam doesn't even test for.
